# Task
*嚴格執行* 請利用你的**聯網瀏覽功能**，搜尋過去 24 小時內（若是週末則搜尋上週五收盤至目前的週末總結）與「美股市場」高度相關的權威財經新聞（如 Bloomberg, CNBC, WSJ, Reuters 等）。
**系統參考時間**：{{CURRENT_DATE}}

請嚴格執行以下 3 個步驟來確保資料的時效性：

1.  **廣泛搜尋與排序**：
    請先聯網搜尋與「美股市場」相關的所有最新權威財經新聞。
    搜尋後，請將所有找到的資料**按時間倒序排列** (最新的在最前面)。

2.  **時間比對與過濾**：
    將排序後的資料時間與上述的 **系統參考時間 ({{CURRENT_DATE}})** 進行比對。
    **只保留** 系統參考時間 **前 3 天內** 的資料。
    任何超過 3 天的舊新聞，請直接捨棄，不予採用。

3.  **生成報告**：
    僅使用經過第 2 步過濾後、確認為 **3 天內** 的最新資料來撰寫這份報告。
    如果搜尋結果顯示是今天的新聞，請直接引用。

搜尋重點應包含：
1. **總體經濟數據**：CPI, PPI, 非農就業, GDP, 10年期美債殖利率, VIX 恐慌指數。
2. **貨幣政策**：Fed 官員談話、利率決策預期。
3. **關鍵事件**：科技巨頭（Magnificent 7）動態、可能影響美股的國際地緣政治、重大併購、原物料價格劇烈波動。
